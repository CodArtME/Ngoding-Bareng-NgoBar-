About

Code Art adalah Suatu organisasi / sekumpulan orang yang mengembangkan bakat dan kemampuannya, dengan cara menuangkan hobby mereka baik dalam bidang programer(code) maupun bidang designer(art). Code Art disini berarti menggabungkan antara code dan art sehingga meciptakan suatu karya / produk yang dapat bermanfaat untuk mempermudah aktifitas dalam kehidupan sehari-hari.

Dalam Organisasi kami membuka pemesanan dari clien / konsumen untuk berbisnis ataupun dalam inspirasi project untuk suatu perusahaan yang diinginkan oleh pengusaha. Adapun kategori yang kami sediakan antara lain :
1. Design Web
2. Web Marketing
3. Form pendaftaran
4. Blogging
5. Website Resmi, Dll.
